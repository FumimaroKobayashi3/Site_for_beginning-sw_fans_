# SW Terminal v1.0 — STAR_WARS_DB 

Проект сделан как экзамен так что не обессуйте

 Описание
Это легкое веб-приложение, имитирующее работу компьютерного терминала из мира Star Wars. Проект подключается к открытому API и позволяет находить информацию о персонажах и планетах.

  Технологии
HTML5 / CSS3: Стилизация под старый ЭЛТ-монитор (эффект скан-линий, свечение текста, моноширинные шрифты).
API: Используется [SWAPI.dev](https://swapi.dev/) для получения актуальных данных.

 Основные функции
 Атмосферный UI: Визуальные эффекты "сканирования галактики" и терминальная эстетика.
Адаптивность: Приложение корректно отображается на разных экранах.

 🛠 Как запустить
Для работы приложения не требуется установка дополнительных зависимостей или запуск сервера:
1. Склонируйте репозиторий или скачайте файлы `index.html` и `main.js`.
2. Откройте файл `index.html` в любом современном веб-браузере.
